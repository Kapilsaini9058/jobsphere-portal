# 🚀 JobSphere - Job Portal Web App

JobSphere is a modern job portal web application built using **React.js**.  
It helps users browse, search, and apply for jobs easily.

---

## 🌐 Live Demo
👉 https://jobportaldevelopbykapil.netlify.app/

---

## 📌 Features

- 🔍 Job search functionality
- 🧑‍💼 Browse job listings
- 📄 Detailed job description pages
- ❤️ Save favorite jobs
- 📱 Responsive design (Mobile + Desktop)
- ⚡ Fast and lightweight UI

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Node.js (for dependencies)
- Netlify (Deployment)
